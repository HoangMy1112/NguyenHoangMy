# My Express App


## Setup

1. Install dependencies:
    ```bash
    npm install
    ```
2. Compile TypeScript:
    ```bash
    npm run build
    ```
3. Start the server:
    ```bash
    npm start
    ```

## API Endpoints

- POST `/books`: Create a book
- GET `/books`: List all books
- GET `/books/:id`: Get a book by ID
- PUT `/books/:id`: Update a book by ID
- DELETE `/books/:id`: Delete a book by ID
